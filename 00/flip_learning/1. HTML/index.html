5. Div와 Span

학교 강좌 사이트를 보면 어떤 영역은 흰색이고 어떤 영역은 파란색이고.. 이렇게 되어 있는데 영역이 구분되어 있으므로 가능한 것이다.
영역은 다음과 같이 분리할 수 있다.

<div>
영역을 새로 지정하고 싶은 내용
</div>

만약 한 줄의 특정 부분만 영역을 지정해주고 싶으면,

<span>
영역을 새로 지정하고 싶은 한 줄의 일부분
</span>

대충 이렇게 몇 문장을 정해 div를 사용할 수 있고..
대충 엄청 긴 논문 한문장에서 span을 사용해 핵심 키워드 영역을 분리할 수 있다.
물론 CSS를 적용하지 않으면 때문에 티가 나지 않는다.

6. 배경 색

이제 div에 배경 색깔을 넣어줄 수 있다!
지정한 div에 대해 <div>를 <div style = "적용될 영역: 색깔">로 작업이 가능하다.

배경색(background-color)을 바꿔야 한다고 가정하자. 그러면 #16진수 RGB, 지정된 용어, rgb()를 사용하여
<div style = "background-color: purple">이나,
<div style = "background-color: C8FFC8">이나,
<div style = "background-color: C8FFC8">처럼 해줄 수 있다.

7. 글자 색

9. 글자 크기

글자 크기도 sytle에서 바꿀 수 있다. 마찬가지로 <div style = "color: 색깔; font-size: 20; ... ">처럼 ;을 사용해 sytle에 이어 붙일 수 있다.
글자가 커지는 건 font-size, 굵어지는 건 font-weight이다.

다음과 같이 <p style = " ... ; font-size: 20">를 단락에 적용하고, 강조할 부분을 < span style = "font-weight: 700">로 처리하면..

초반에 힘이 약했던 나서스는 1차 타워를 밀 때쯤 q 스택이 많이 쌓여 그럴듯한 딜이 나오지만, 극후반이 되면 이동기가 없어 힘이 빠진다.

같은 그럴 듯한 강조를 할 수 있다.

10. 글자 정렬

글자를 정렬하는 것도 sytle에서 바꿀 수 있다. <div style = " ... ; font-align : 위치">처럼 앞선 예제와 동일하게 ;을 사용하여 sytle에 이어 붙일 수 있다.
위치는 left, center, right가 존재한다. 각각 왼쪽/중앙/오른쪽 정렬이다.



이렇게 잘 중앙으로 정렬된 글이.. "text-align: center"
왼쪽으로 몰아넣을 수 있고.. "text-align: left"
마찬가지로 오른쪽도 가능하다. "text-align: right"

11. 문서의 크기

사진과 같은 개체부터 문단의 크기까지 width, height를 사용하여 조절할 수 있다. <div style = " ... ; width = 크기; height = 크기">처럼 sytle에 이어 붙일 수 있다.
크기에는 고정된 값인 px, 비율인 %가 들어갈 수 있다. 각각 460px와 100%와 같은 표현이 가능하다.
참고로 사진에 이어 붙이려면 <img scr = "링크" style = "width: ...> 처럼 src와 width 사이에 공백( , 가 아님!!!!)으로 분리해주면 된다.

예를 들어서, 아까 적었던 글자 정렬을 활용하면 다음과 같은 표현을 할 수 있다. 전체 문단의 width를 100px로 설정해..

< div style = "color:aqua; width: 100% >가 적용되었습니다.
< div style = "color:aqua; width: 80% >가 적용되었습니다.
< div style = "color:aqua; width: 60% >가 적용되었습니다.
< div style = "color:aqua; width: 40% >가 적용되었습니다.

라고 할 수 있다. 창 크기를 크게 줄이면 차이점이 보일 것이다.

12. 문서의 여백

<div style = " ... ; padding: ypx, xpx";처럼 sytle에 이어 붙일 수 있다.
ypx는 세로 여백 크기, xpx는 가로 여백 크기이다.

본 문서의 전체에서는 세로 여백 40, 가로 여백 20으로 설정하였다. 즉, <body style = "padding: 40px 20px">가 들어간 것이다.
